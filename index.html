About

Gemu.co.io is a personal project by Himanshu Pal, created as a fun and creative space to showcase mini-games built using HTML, CSS, and JavaScript. Each game on this site is handcrafted with passion for coding, design, and interactive experiences. The goal is to blend learning with entertainment — whether you're here to play, get inspired, or explore what’s possible with frontend development. New games are added regularly, so stay connected and enjoy the journey!
